Statistics of the agents

  • Train loss: Training loss is the error on the training set of data
  • Val loss: Validation loss is the error after running the validation set of data through the trained network
  • Train acc: The accuracy of a agent on examples it has seen while training
  • Val acc: The accuracy of the agent from images that is has not seen while training
